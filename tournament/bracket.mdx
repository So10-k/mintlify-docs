The bracket page gives you a visual map of the entire tournament. Each box is a matchup between two players; winners advance to the next column on the right until only one player remains.

## Main bracket

The main bracket is a single-elimination draw. Every round, each remaining player faces one opponent. The higher scorer in their quiz round advances to the next round; the lower scorer is knocked out (or drops to the losers bracket if one exists — see below).

Matchup results are filled in automatically after each quiz round closes. The bracket updates in real time as round scores are processed, so you can watch it fill in as the tournament progresses.

The final matchup on the right side of the main bracket shows the **Champion** slot. Once the final round resolves, the champion's name appears there.

<Tip>
  You can follow your own path through the bracket by tracing your name from Round 1 forward.
</Tip>

## Losers bracket

Some tournaments include a losers bracket for players who were knocked out in Round 1. If the host has enabled this, a second bracket section appears below the main bracket.

<CardGroup cols={2}>
  <Card title="Who enters" icon="heart-crack">
    Only players eliminated in Round 1 of the main bracket drop into the losers bracket. Players knocked out in later main-bracket rounds are fully eliminated.
  </Card>
  <Card title="Last chance" icon="rotate-ccw">
    The losers bracket is a second-chance draw. Winners from it do not re-enter the main bracket — the losers bracket determines its own final outcome separately.
  </Card>
</CardGroup>

If no losers bracket is shown, the host has not enabled this feature for the current tournament.

## How results are decided

Bracket results are set by quiz scores:

- When a round closes, the player with the higher score in that round's quiz advances.
- In the event of a tie, the matchup stays undecided until the host resolves it manually.
- The host can also override any result directly, for example to correct a data issue.

Slots for later rounds appear empty until earlier rounds complete and winners propagate forward.

## Page availability

The host can temporarily hide the bracket for all players. When the page is hidden, you will see a "paused" notice. The bracket becomes visible again once the host unlocks it.

<Info>
  If you see the paused notice, check back after the current round concludes. The host typically reveals the updated bracket shortly after scores are in.
</Info>
